<h2><a href=https://www.codewars.com/kata/5721c189cdd71194c1000b9b/train/javascript target="_blank">Training JS #11: loop statement --break,continue</a></h2><h3>8 kyu</h3><p>We have seen <code>break</code> in the switch statement, it can jump out the switch statement. it can also be used in the loop statement. an example:</p><pre><code class="language-javascript"><span class="cm-keyword">function</span> <span class="cm-def">findFirstOddNumber</span>(<span class="cm-def">arr</span>){  <span class="cm-keyword">var</span> <span class="cm-def">result</span>;  <span class="cm-keyword">for</span> (<span class="cm-keyword">var</span> <span class="cm-def">i</span><span class="cm-operator">=</span><span class="cm-number">0</span>;<span class="cm-variable-2">i</span><span class="cm-operator">&lt;</span><span class="cm-variable-2">arr</span>.<span class="cm-property">length</span>;<span class="cm-variable-2">i</span><span class="cm-operator">++</span>){    <span class="cm-keyword">if</span> (<span class="cm-variable-2">arr</span>[<span class="cm-variable-2">i</span>]<span class="cm-operator">%</span><span class="cm-number">2</span><span class="cm-operator">==</span><span class="cm-number">1</span>){      <span class="cm-variable-2">result</span><span class="cm-operator">=</span><span class="cm-variable-2">arr</span>[<span class="cm-variable-2">i</span>];      <span class="cm-keyword">break</span>;    }  }  <span class="cm-keyword">return</span> <span class="cm-variable-2">result</span>;}</code></pre><p>In this example, for loop traverse the array arr. when the first odd number found, <code>break</code> statement will jump out the for loop.</p><p>In the loop statement, we can use <code>continue</code> skipping some code. for example, the following code ignores negative values:</p><pre><code class="language-javascript"><span class="cm-keyword">function</span> <span class="cm-def">dontDisplayNegative</span>(<span class="cm-def">arr</span>){  <span class="cm-keyword">for</span> (<span class="cm-keyword">var</span> <span class="cm-def">i</span><span class="cm-operator">=</span><span class="cm-number">0</span>;<span class="cm-variable-2">i</span><span class="cm-operator">&lt;</span><span class="cm-variable-2">arr</span>.<span class="cm-property">length</span>;<span class="cm-variable-2">i</span><span class="cm-operator">++</span>){    <span class="cm-keyword">if</span> (<span class="cm-variable-2">arr</span>[<span class="cm-variable-2">i</span>]<span class="cm-operator">&lt;</span><span class="cm-number">0</span>) <span class="cm-keyword">continue</span>;       <span class="cm-variable">console</span>.<span class="cm-property">log</span>(<span class="cm-variable-2">arr</span>[<span class="cm-variable-2">i</span>])  }}</code></pre><p>In this example, <code>console.log(arr[i])</code> is never executed for negative element. thanks to <code>continue</code>.</p><p>Through the use of <code>break</code> and <code>continue</code>, you can make the loop statement more flexible and convenient.</p><p>Ok, lesson is over. let's us do some task with <code>break</code> and <code>continue</code>.</p><h2 id="task">Task</h2><p>Coding in function <code>grabDoll</code>. function accept 1 parameter:<code>dolls</code>. it's a string array, a list of some dolls.</p><p>You need traverse <code>dolls</code> by using <code>for</code> loop. If element is  "Hello Kitty" or "Barbie doll", you should push it to a <code>bag</code>(bag is an array, I've defined in the function); if it's other strings, we should use <code>continue</code> skip it. </p><p>When the <code>bag</code> has three element, <code>bag</code> is full. You should use <code>break</code> jump out the loop; If <code>bag</code> is not full, you should traverse <code>dolls</code> until the last element. </p><p>Return the <code>bag</code> after for loop finished.</p><p>You should use <code>for</code>, <code>break</code> and <code>continue</code> in your code. otherwise, your solution may not pass this kata.</p><p>If you forgot how to push an element

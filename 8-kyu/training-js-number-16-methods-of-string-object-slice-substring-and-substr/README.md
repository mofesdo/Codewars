<h2><a href=https://www.codewars.com/kata/57274562c8dcebe77e001012/train/javascript target="_blank">Training JS #16: Methods of String object--slice(), substring() and substr()</a></h2><h3>8 kyu</h3><p>From this lesson, we begin to learn about String objects. </p><p>First of all, we have a look at the attributes of String: <code>length</code>. Use <code>StringObject.length</code> to get the length of a string, like this:</p><pre><code class="language-javascript"><span class="cm-keyword">let</span> <span class="cm-def">str</span><span class="cm-operator">=</span><span class="cm-string">"dsfsdwesdfsadfwerfsdfsdfsdfsdfsdf"</span>;<span class="cm-variable">console</span>.<span class="cm-property">log</span>(<span class="cm-variable">str</span>.<span class="cm-property">length</span>)  <span class="cm-comment">//output: 33</span></code></pre><p>Secondly, a string can be like an array. each character in a string has an index. We can use the index to access this character. like this:</p><pre><code class="language-javascript"><span class="cm-keyword">let</span> <span class="cm-def">str</span><span class="cm-operator">=</span><span class="cm-string">"codewars"</span>;<span class="cm-keyword">for</span> (<span class="cm-keyword">let</span> <span class="cm-def">i</span><span class="cm-operator">=</span><span class="cm-number">0</span>;<span class="cm-variable-2">i</span><span class="cm-operator">&lt;</span><span class="cm-variable">str</span>.<span class="cm-property">length</span>;<span class="cm-variable-2">i</span><span class="cm-operator">++</span>) <span class="cm-variable">console</span>.<span class="cm-property">log</span>(<span class="cm-variable">str</span>[<span class="cm-variable-2">i</span>])<span class="cm-comment">//output:</span><span class="cm-variable">c</span><span class="cm-variable">o</span><span class="cm-variable">d</span><span class="cm-variable">e</span><span class="cm-variable">w</span><span class="cm-variable">a</span><span class="cm-variable">r</span><span class="cm-variable">s</span></code></pre><p>String Object has a lot of methods, many of these methods are used for web page text style and color modification. These methods are rarely used in programming and some of they are deprecated now. So we will omit the learning of these methods. These methods are:</p><pre><code>anchor()    Create HTML anchor.big()       Display a string in a large font.blink()     Display flashing string.bold()      Use bold display string.fixed()     Using a typewriter text display string.fontcolor() Displays a string using the specified color.fontsize()  Displays a string using the specified size.italics()   Use italics string.link()      Display a string as a link.small()     Use small size to display string.strike()    Use the delete line to display the string.sub()       Display the string as a subscript.sup()       Display the string as a superscript.</code></pre><p>This time we learn three methods that used to obtain the substring of the string:<code>slice()</code> <code>substring()</code> and <code>substr()</code>. Their usage:</p><pre><code class="language-javascript"><span class="cm-variable">StringObject</span>.<span class="cm-property">slice</span>(<span class="cm-variable">startindex</span>,<span class="cm-variable">endindex</span>)<span class="cm-variable">StringObject</span>.<span class="cm-property">substring</span>(<span class="cm-variable">startindex</span>,<span class="cm-variable">endindex</span>)<span class="cm-variable">StringObject</span>.<span class="cm-property">substr</span>(<span class="cm-variable">startindex</span>,<span class="cm-variable">length</span>)</code></pre><p>Here we use some examples to illustrate their usage and difference:</p><pre><code class="language-javascript"><span class="cm-keyword">let</span> <span class="cm-def">str</span><span class="cm-operator">=</span><span class="cm-string">"Hello World!"</span>;<span class="cm-variable">console</span>.<span class="cm-property">log</span>(<span class="cm-variable">str</span>.<span class="cm-property">slice</span>(<span class="cm-number">6</span>))<span class="cm-variable">console</span>.<span class="cm-property">log</span>(<span class="cm-variable">str</span>.<span class="cm-property">substring</span>(<span class="cm-number">6</span>))<span class="cm-variable">console</span>.<span class="cm-property">log</span>(<span class="cm-variable">str</span>.<span class="cm-property">substr</span>(<span class="cm-number">6</span>))<span class="cm-comment">//output:</span><span class="cm-variable">World</span><span class="cm-operator">!</span><span class="cm-variable">World</span><span class="cm-operator">!</span><span class="cm-variable">World</span><span class="cm-operator">!</span></code></pre><p>We can see, the first parameter startindex can not be omitted, it represents the beginning of the interception of the string. Second parameter can be omitted, the three method will intercept the string to the end of the string. Look at the following example:</p><pre><code class="language-javascript"><span class="cm-keyword">let</span> <span class="cm-def">str</span><span class="cm-operator">=</span><span class="cm-string">"Hello World!"</span>;<span class="cm-variable">console</span>.<span class="cm-property">log</span>(<span class="cm-variable">str</span>.<span class="cm-property">slice</span>(<span class="cm-number">0</span>,<span class="cm-number">5</span>))<span class="cm-variable">console</span>.<span class="cm-property">log</span>(<span class="cm-variable">str</span>.<span class="cm-property">substring</span>(<span class="cm-number">0</span>,<span class="cm-number">5</span>))<span class="cm-variable">console</span>.<span class="cm-property">log</span>(<span class="cm-variable">str</span>.<span class="cm-property">substr</span>(<span class="cm-number">0</span>,<span class="cm-number">5</span>))<span class="cm-comment">//output:</span><span class="cm-variable">Hello</span><span class="cm-variable">Hello</span><span class="cm-variable">Hello</span></code></pre><p>We can see, When the startindex is 0, the results obtained by the three methods are the same. But in fact, their operations are different. <code>slice()</code> and <code>substring()</code> intercepted characters from index0 to index5 (not including index5). substr  operation is to start from the index0, the interception of a substring of length 5. Look at the following example:</p><pre><code class="language-javascript"><span class="cm-keyword">let</span> <span class="cm-def">str</span><span class="cm-operator">=</span><span class="cm-string">"Hello beautiful World!"</span>;<span class="cm-variable">console</span>.<span class="cm-property">log</span>(<span class="cm-variable">str</span>.<span class="cm-property">slice</span>(<span class="cm-number">6</span>,<span class="cm-number">16</span>))<span class="cm-variable">console</span>.<span class="cm-property">log</span>(<span class="cm-variable">str</span>.<span class="cm-property">substring</span>(<span class="cm-number">6</span>,<span class="cm-number">16</span>))<span class="cm-variable">console</span>.<span class="cm-property">log</span>(<span class="cm-variable">str</span>.<span class="cm-property">substr</span>(<span class="cm-number">6</span>,<span class="cm-number">9</span>))<span class="cm-comment">//output:</span><span class="cm-variable">beautiful</span> <span class="cm-variable">beautiful</span> <span class="cm-variable">beautiful</span></code></pre><p>We can see, In order to intercept the same string "beautiful", they use different parameters. Looks like substring and slice usage is similar, what is the difference between them? Look at the following example:</p><pre><code class="language-javascript"><span class="cm-keyword">let</span> <span class="cm-def">str</span><span class="cm-operator">=</span><span class="cm-string">"Hello beautiful World!"</span>;<span class="cm-variable">console</span>.<span class="cm-property">log</span>(<span class="cm-variable">str</span>.<span class="cm-property">slice</span>(<span class="cm-number">16</span>,<span class="cm-number">21</span>))<span class="cm-variable">console</span>.<span class="cm-property">log</span>(<span class="cm-variable">str</span>.<span class="cm-property">substring</span>(<span class="cm-number">16</span>,<span class="cm-number">21</span>))<span class="cm-variable">console</span>.<span class="cm-property">log</span>(<span class="cm-variable">str</span>.<span class="cm-property">slice</span>(<span class="cm-number">16</span>,<span class="cm-operator">-</span><span class="cm-number">1</span>))<span class="cm-comment">//output:</span><span class="cm-variable">World</span><span class="cm-variable">World</span><span class="cm-variable">World</span></code></pre><p>From this example we can see that the parameters of the <code>slice()</code> can be a negative number. Negative meaning is starting from the right. -1 is first position from the right(at the end of that string).</p><p>In a word, for <code>silce()</code> and <code>substring()</code>, we usually choose to use slice(), because it is more flexible and simple.
